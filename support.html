常见问题

查找常见问题的解答，快速解决您在使用JETFAY产品时可能遇到的问题。

遥控器

如何校准遥控器摇杆？

校准遥控器摇杆的步骤如下：

  1. 确保遥控器处于关机状态
  2. 按住两个摇杆中心位置，同时打开遥控器电源
  3. 遥控器进入校准模式后，LED指示灯会闪烁
  4. 按照屏幕提示（或LED指示），将两个摇杆分别移动到最大范围的四个方向
  5. 完成后，按下确认按钮保存校准结果

如果您的遥控器型号支持通过PC软件校准，也可以连接电脑，使用JETFAY遥控器配置软件进行更精确的校准。

遥控器无法与接收机对频怎么办？

如果遥控器无法与接收机对频，请尝试以下解决方法：

  1. 确保接收机已正确供电，并且LED指示灯正常工作
  2. 检查遥控器和接收机是否为同一系列产品，不同系列可能不兼容
  3. 重新进行对频操作：
    • 按住接收机上的对频按钮，接收机LED开始快速闪烁
    • 在遥控器上进入对频模式（通常是按住特定按钮或组合键）
    • 当接收机LED变为常亮时，表示对频成功
  4. 如果仍然无法对频，尝试更新遥控器和接收机的固件到最新版本
  5. 检查周围环境是否有强电磁干扰，尝试在不同位置进行对频

如果以上方法都无效，可能是硬件问题，请联系JETFAY技术支持。

如何自定义遥控器按键功能？

自定义遥控器按键功能需要使用JETFAY遥控器配置软件，步骤如下：

  1. 下载并安装最新版本的JETFAY遥控器配置软件
  2. 使用USB数据线连接遥控器和电脑
  3. 打开配置软件，软件会自动识别您的遥控器型号
  4. 进入"按键设置"或"功能映射"页面
  5. 选择要自定义的按键，从下拉菜单中选择所需功能
  6. 根据需要调整参数（如长按/短按行为、组合键功能等）
  7. 点击"保存"或"写入"按钮，将设置写入遥控器

不同型号的遥控器支持的自定义功能可能有所不同，请参考产品手册了解您的遥控器支持的具体功能。

飞控系统

如何调整PID参数？

调整PID参数是优化飞行性能的关键步骤，建议按以下流程进行：

  1. 使用JETFAY飞控配置软件连接飞控
  2. 进入PID调参页面，查看当前参数
  3. 调整P值（比例）：
    • 从较低的值开始，逐步增加直到飞机反应灵敏但不过冲
    • 如果出现振荡，则适当降低P值
  4. 调整I值（积分）：
    • I值主要用于消除稳态误差，保持飞机稳定
    • I值过高会导致飞机反应迟缓，过低则无法保持稳定
  5. 调整D值（微分）：
    • D值用于抑制过冲和振荡
    • D值过高会导致高频振荡，过低则无法抑制过冲
  6. 每次调整一个参数，并进行飞行测试
  7. 使用黑盒记录功能记录飞行数据，分析调整效果

建议初学者从厂商推荐的默认值开始，小幅调整，避免大幅度改变参数。不同的机型和飞行风格需要不同的PID参数，需要耐心调试。

飞控无法连接电脑怎么办？

飞控无法连接电脑可能有多种原因，请按以下步骤排查：

  1. 检查USB连接：
    • 尝试更换USB数据线，确保使用的是数据线而非仅充电线
    • 尝试不同的USB接口，避免使用USB集线器
  2. 检查驱动程序：
    • 确保已安装正确的USB驱动程序
    • 在设备管理器中查看是否有未识别的设备或驱动问题
  3. 检查飞控电源：
    • 确保飞控通过USB或外部电源正确供电
    • 某些飞控需要外部电源才能通过USB进行配置
  4. 检查配置软件：
    • 确保使用的是最新版本的JETFAY飞控配置软件
    • 检查软件中的COM端口设置是否正确
  5. 尝试恢复出厂设置：
    • 按住飞控上的复位按钮，同时接通电源
    • 具体操作方法请参考产品手册

如果以上方法都无效，可能是飞控硬件损坏或固件损坏，请联系JETFAY技术支持。

电子调速器

电调发出异常蜂鸣声是什么原因？

电调发出异常蜂鸣声通常是在提示某种错误或警告状态，常见原因和对应的蜂鸣模式如下：

  • 无信号警告：通常是连续短促的蜂鸣声，表示电调未接收到正确的油门信号。检查接收机连接和遥控器设置。
  • 电池电压异常：通常是长短交替的蜂鸣声，表示电池电压过高或过低。检查电池电压和电调的电压设置。
  • 启动保护：快速连续的蜂鸣声，表示在油门不为零的情况下接通电源。将油门拉到最低位置。
  • 过热保护：缓慢的长音蜂鸣，表示电调温度过高。停止使用，等待冷却，并检查散热情况。
  • 过流保护：急促的长短交替蜂鸣，表示电流超过电调额定值。检查电机是否卡住或负载过重。

不同型号的电调可能有不同的蜂鸣提示模式，请参考产品手册了解具体的蜂鸣代码含义。如果蜂鸣持续存在且无法解决，可能是电调内部损坏，需要联系技术支持或更换电调。

如何校准电调？

校准电调的一般步骤如下（具体操作可能因电调型号而异）：

  1. 断开电调与电池的连接
  2. 打开遥控器，将油门摇杆推到最高位置
  3. 连接电调与电池，等待约2秒
  4. 电调会发出确认音（通常是上升音调）
  5. 将油门摇杆拉到最低位置
  6. 电调会发出确认音（通常是下降音调），表示校准完成
  7. 断开电池，然后重新连接，电调现在应该正常工作

对于支持编程的高级电调，也可以使用JETFAY电调编程卡或编程软件进行更精确的校准和参数设置。校准前请确保已阅读产品手册，了解您的电调型号的具体校准方法。

一般问题

如何注册产品保修？

注册JETFAY产品保修的步骤如下：

  1. 访问JETFAY官方网站的保修注册页面
  2. 创建JETFAY用户账户或登录已有账户
  3. 点击"注册新产品"按钮
  4. 输入产品序列号（通常在产品或包装上）
  5. 上传购买凭证（发票或收据的扫描件/照片）
  6. 填写必要的个人信息和购买信息
  7. 提交注册表单

成功注册后，您将收到确认邮件，并可以在用户中心查看产品保修状态。建议在购买后30天内完成产品注册，以确保获得完整的保修服务。

如何查询产品真伪？

查询JETFAY产品真伪的方法有以下几种：

  1. 防伪码验证：
    • 每个JETFAY正品都有唯一的防伪码
    • 访问JETFAY官网的产品验证页面
    • 输入产品防伪码进行验证
  2. 微信验证：
    • 关注JETFAY官方微信公众号
    • 进入"产品验证"菜单
    • 扫描产品包装上的二维码或输入防伪码
  3. APP验证：
    • 下载JETFAY官方APP
    • 登录后进入"产品验证"功能
    • 扫描产品包装上的二维码或输入防伪码

如果验证结果显示产品不是正品，请立即联系您的购买渠道或JETFAY客服。购买JETFAY产品时，建议选择官方授权的销售渠道，避免购买到假冒产品。

产品需要维修怎么办？

当您的JETFAY产品需要维修时，请按以下步骤操作：

  1. 确认保修状态：
    • 登录JETFAY官网用户中心查看产品保修状态
    • 确认产品是否在保修期内，以及故障类型是否属于保修范围
  2. 联系客服：
    • 拨打客服热线400-888-XXXX
    • 或发送邮件至support@jetfay.com
    • 描述产品型号、故障现象和购买信息
  3. 获取RMA号：
    • 客服会为您提供一个RMA（返修授权）号码
    • 记录此号码，用于后续跟踪维修进度
  4. 寄送产品：
    • 按照客服提供的地址寄送产品
    • 包装中附上RMA号码、故障描述和联系方式
    • 建议使用可跟踪的快递方式
  5. 维修进度查询：
    • 通过官网"维修查询"页面输入RMA号码查询
    • 或联系客服查询维修进度

维修完成后，产品将被寄回您提供的地址。保修期内的产品维修通常是免费的（人为损坏除外），保修期外的维修将收取一定的维修费用。
